Search and Storage of Olympiads
-

1 Что? Что вы собираетесь делать?
-
Я совместно с Центром Талантов Кемерово разрабатываю 1 МОДУЛЬ БОЛЬШОГО ПРОЕКТА для поиска олимпиад и сохранения учениками своих результатов по их итогам в список. Суть МОДУЛЯ КОТОРЫЙ СЕЙЧАС РАЗРАБАТЫВАЕТСЯ - это искать олимпиады. 

Процесс работы модуля: 

1.Пользователю выводятся фильтры для олимпиад (темы, направления, формат проведения - онлайн/офлайн,)

2.Затем программа производит парсинг сайтов (выпускающие списки олимпиад)

3.Выводит список олимпиад и ссылки на основные сайты олимпиад с инф.

2 Зачем? \ почему? Зачем этот продукт или изделие будут покупать? Какую пользу оно может дать
-
Смысл покупки проекта – это автоматизация процесса поиска олимпиад и удобное хранение своих результатов. В место того, чтобы вручную смотреть сайты с олимпиадами (искать их основные сайты для регистрации и т.д.), одним выбором фильтра будет показан список олимпиад с ссылками на их основные сайты (для дальнейшего просмотра пользователем). Это банально экономит время. А после участия можно будет сохранить свой результат (добавив его к списку олимпиад) 

3 Где? Где вы собираетесь реализовывать и продавать ваше изделие?
-
Я собираюсь реализовать проект в виде отдельной вкладки на новом сайте Центра Талантов Кемерово (если всё будет нормально работать; Семён и др. допустят до этой стадии). На данный момент планируется тестирования на своём сайте и только потом бесплатное тестирование на новом сайте Центром Талантов Кемерово, дальнейшее продвижение проекта будет продуманно после удачного тестирования. Сама продажа продукта будет на сайте (на котором будет 1 тестирование) в формате подписки, как для обычных пользователей, так и для образовательных учреждений (Freemium – для Центра Талантов Кемерово; Premium – для учеников и преподавателей образовательного учреждения, купившего соответственную подписку; Стандартная подписка – для обычных пользователей).

4 Когда? Будет ли сезонность выпуска продукта? Когда вы сможете продавать ваш продукт? Ситуация, которая приведёт к покупке?
-
Для работы проекта сезонности нет. Но нагрузка и частота использования целиком зависят от олимпиад. Пик по используемости и нагрузки - период с сентября по декабрь. Ситуация, которая приведёт к покупке - автоматизации поиска олимпиад (банально экономит время) и сохранение результатов (ты не потеряешь свои достижения, сейчас потерял бумажку – и всё ☹). 

5 Кто? Кто будет покупать? Целевая аудитория?
-
Целевой аудиторией являются образовательные учреждения (на данный момент и в первую очередь Центр Талантов Кемерово), в дальнейшем будет продуманна система для продвижения на массовую аудиторию. Образовательные учреждения будут покупать продукт из-за автоматизации поиска всех проводимых (и прошедших) олимпиад, ссылки на их основные сайты для более конкретного изучения (банально экономит время) и сохранение результатов учеников. Это посредствует повышению качества образования, улучшению репутации, привлечению финансирования, развитию инфраструктуры и программ, мотивации преподавателей и учеников.

6 Какая бизнес-модель (b2b, b2g, b2c)?
-
На данный момент – это b2b, т.к. проект будет находиться в стадии тестирования на новом сайте Центром Талантов Кемерово. Соответственно он направлен на продвижение по другим Центрам Талантов, что является сотрудничеством с Фондом Мельниченко и по его рекомендации будет возможно расширение на другие образовательные учреждения. В дальнейшем можно продвигать проект в модели b2c, для массового распространения (с помощью стандартных подписок). И в модели b2g, для сотрудничества с государством и возможным продвижением на иностранные рынки (с помощью Premium подписки).
